Route Builder v1.0

Flask + Leaflet.js + OSRM

Web App v1.0.0

FC 셔틀 노선 빌더

물류센터(FC) 셔틀버스 노선 생성 시스템. 41개 FC, 4,274개 정류장 커버.

Core Features

  • 클릭으로 정류장 추가 (인터랙티브 맵)
  • 75분 노선 + 44인승 제약 실시간 검증
  • OSRM + Kakao API 하이브리드 라우팅
  • FC 도착 시간 기반 자동 시간표 생성

Technical Workflow

1

FC 선택 및 정류장 로드

41개 물류센터 중 선택 시 반경 내 4,274개 정류장이 지도에 표시됩니다.

2

클릭으로 노선 구성

정류장 클릭 순서대로 노선이 구성되며, OSRM이 실시간으로 도로 기반 경로를 계산합니다.

3

제약조건 실시간 검증

75분 운행시간 및 44인승 버스 정원 초과 시 경고가 표시됩니다.

4

시간표 자동 생성

FC 도착 시간을 기준으로 역산하여 각 정류장 출발 시간표가 자동 생성됩니다.
